# timetracker
Have you ever thought of time is not enough?

I'm a undergraduate and I felt it. After a few consideration, I came to a conclusion. 

The conclusion: I spent too much time on academics. 

Ever wonder how much time spent on academics, especially just for the coursework, assignments and stuffs. I agree that we have to study but never too much.
Therefore, I'm creating this 'SaveTime', using this app, we can feel exactly how much time we have spent on 'which, what' subjects. 
Most importantly, we can prevent overstudy with this.
